# Git 入门

# git 操作的具体步骤

  1. cd 路径 进入当前目录

  2. 配置 git 基本操作

    【注】没有消息就是好消息。

  3. git init 在本地进行初始化 (建立暂存区)

    .git 文件存储当前项目的所有版本信息

  4. 工作区 => 暂存区

    git add 文件名

    git add * 提交所有文件

    git commit -m "这一次提交的描述"

  5. 查看当前工作区的状态

    git status

  6. 从暂存区恢复文件到工作区

    git checkout 文件名

  7. 查看工作区和暂存区版本的区别

    git diff

  8. clear 清屏操作

  9. git log 查看已经提交到暂存区的历史版本

  10. git reset --hard 版本号 恢复文件到指定的某一个版本

  11. ssh-keygen -t rsa –C "你github邮箱" 生成 ssh 密匙

    windows 电脑查找文件：我的电脑 => 用户 => 用户名文件夹 => .ssh (隐藏文件) => id_rsa.pub (存有密匙)

  12. github 账户配置密匙

  13. 暂存区 => 远程仓库

    git remote add origin https://github.com/xxx/xxx.git
    git push -u origin master

    【注】提交过程，可能需要你输入用户名，密码。

  14. git clone 从远程仓库，克隆项目到本地

  15. git pull 从远程仓库同步本地代码 (更新)
