# 笔记

# 渐变

  1. 线性渐变 -> linear-gradient 是值，需要添加到 background-image 属性上

    linear-gradient：线性渐变（默认上往下）

    • point 指定目标点
    • angle 指定角度
    • color （逗号隔开）
    • percentage（百分比）

    注：渐变的 0 度是在页面在下边，正值会按照顺时针旋转，负值按逆时针旋转。

  2. 径向渐变 -> radial-gradient

    • point
    • color
    • percentage

# 字体图标

font-face 是 CSS3 中的一个模块，他主要是把自己定义的 Web 字体嵌入到你的网页中。

  • 好处：
  1. 可以非常方便的改变大小和颜色
    font-size color
  2. 放大后不会失真
  3. 减少请求次数和提高加载速度
  4. 简化网页布局
  5. 减少设计师和前端工程师的工作量
  6. 可使用计算机没有提供的字体

  • 使用：

    @font-face 语法

    像.woff 等文件都是做兼容平台处理的，mac、linux 等。

# 文字阴影、盒子阴影

  • text-shadow（文字阴影）

    x y blur（模糊值） color

    注：阴影的默认颜色是跟文字样色相同
    注：通过逗号的方式隔开，可以设置多重阴影

  • box-shadow（盒子阴影）

    x y blur spread（范围） color inset（内阴影） 多阴影

    注：盒子阴影的默认样色是黑色。
    注：默认外阴影，设置 outset 则不起作用，可选的值只有 inset 表示内阴影。
