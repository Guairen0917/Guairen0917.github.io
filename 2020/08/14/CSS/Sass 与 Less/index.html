# Sass 与 Less

​ Sass 和 Less 都属于 CSS 预处理器，CSS 预处理器定义了一种新的语言，其基本思想是，用一种专门的编程语言，为 CSS 增加了一些编程的特性，如：变量、语句、函数、继承等概念。将 CSS 作为目标生成文件，然后开发者就只要使用这种语言进行 CSS 的编码工作。

语法：注释
变量，插值，作用域
选择器嵌套，伪类嵌套，属性嵌套（Sass）
运算，单位，转义，颜色函数
混入，命名空间（Less），继承合并，媒体查询
条件，循环
导入

# PostCSS

​ PostCSS 本身是一个功能比较单一的工具。它提供了一种方式用 JavaScript 代码来处理 CSS。利用 PostCSS 可以实现一些工程化的操作，如：自动添加浏览器前缀，代码合并，代码压缩等。

官方网址：https://postcss.org/

postcss src/demo.css -o dist/demo.css    //转化
postcss src/demo.css -o dist/demo.css -w //监听

  • postcss.config.js 配置文件

  • 常见插件：

    autoprefixer // 添加浏览器前缀

    postcss-import // 对 css 文件合并操作

    cssnano // 对 css 文件压缩操作

    postcss-cssnext // 降级

    stylelint // 书写规范

    postcss-sprites // 生成精灵图
