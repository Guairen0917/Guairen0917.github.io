# BOM & DOM

JavaScript组成 ： ECMAScript BOM DOM
ECMAScript是一种语法标准语法:变量和数据类型、运算符、逻辑控制语句、关键字、保留字对象
编码遵循ECMAScript标准
BOM:Browser Object Model (浏览器对象模型) 提供独立于内容与浏览器窗口进行交互的对象
DOM:Document Object Model (文档对象模型) 是HTML文档对象模型(HTML DOM) 定义的一套标准方法，用来访问和操作HTML文档

# 正则表达式（Regular Expression）

概念：
	正则表达式使用单个字符串来描述、匹配一系列符合某个句法规则的字符串搜索模式
	用某种模式去匹配一类字符串的公式
作用：通过使用正则表达式，可以：
	测试字符串内的模式
	替换文本
	从字符串中提取子字符串
语法：					   正则函数("表达式","修饰符")
	构造函数：var reg = new RegExp(pattern,modifiers);
	字面量：  var reg =/pattern/modifiers;
	修饰符： 
		i	执行对大小写不敏感的匹配
		g	执行全局匹配（查找所有匹配而非在找到第一个匹配后停止）
		m	执行多行匹配

# 正则表达式方法

  • 检测一个字符串是否与正则相匹配

    reg.test(string);  // 返回值为布尔值 true 匹配, false 不匹配
reg.exec(string);  // 匹配成功返回数组，并确定位置，否则返回 null

  • String 类中支持的正则方法：

    search		检索与正则表达式相匹配的值		返回下标，否则返回 -1
match		找到一个或多个正则表达式的匹配    返回数组(全局 g)，否则返回 null
replace		替换与正则表达式匹配的子串		返回新字符串 (可全局 g)
split		把字符串分割为字符串数组。		返回字符串分割数组

# 创建对象 （4 种）

  1. Object
ECMAScript提供的一种原生引用类型
可以通过Object引用类型的实例创建一个对象
可以为对象定义属性和方法
Object实例本身不具备多少功能，一般在应用程序中存储和传输数据
  1. 对象字面量
  	对象定义的一种简写形式
  	简化创建包含大量属性的对象的过程
  	在为函数传递大量可选参数时，可考虑使用对象字面量
  1. 工厂模式
    软件工程领域的一种设计模式
	抽象了创建对象的过程
	通过函数封装创建对象的细节
  1. 构造函数
	构造函数一般以大写字母开头
	构造函数也是函数，只不过可以用来创建对象
	与工厂模式对比：
		没有显式创建对象
		直接将属性和方法赋给了 this 对象
		没有 return
  1. 原型 prototype
每个函数都有一个prototype）（原型）属性
是一个指针，指向一个对象
这个对象的用途是包含可以由特定类型的所有实例共享的属性和方法
  1. 混合方式构造对象
构造函数 + prototype
  构造函数：属性
  原型protptype：方法

# 继承

# 原型链继承

  • prototype 属性

prototype 属性包含一个对象（prototype 对象），
所有实例对象需要共享的属性和方法，都放在这个对象里，
实例对象一旦创建，将自动引用（共享）prototype 对象的属性和方法，
prototype 对象好像是实例对象的原型，而实例对象则好像 "继承" 了 prototype 对象一样

  • 直接继承 prototype

是对上一种方法的改进。
由于 Person 对象中，不变的属性都可以直接写入 Person.prototype。所以，可以让 Student () 跳过 Person ()，直接继承 Person.prototype。
优点：相较于第一种，不用执行和建立 Person 实例，节省内存
缺点：父类 Person 的 constructor 也改掉了

  • 利用空对象作为中介

空对象，几乎不占内存
修改 Student 的 prototype 对象，不会影响到 Person 的 prototype 对象（解决继承链紊乱问题）
可以封装为函数

  • 构造函数绑定
在子类型构造函数的内部调用父类型构造函数
通过call()或apply()方法 -> 两方法产生的作用相同，传参不同
  function.call(thisObj[,arg1[,arg2[,[,...argN]]]])
  调用一个对象的一个方法，以另一个对象替换当前对象。
  例如：B.call(A, args1, args2); 即A对象调用B对象的方法

function.apply(thisObj[,argArray])
应用某一对象的一个方法，用另一个对象替换当前对象。
例如：B.apply(A, arguments); 即A对象应用B对象的方法
  • 组合继承（伪经典继承）

将 原型链继承 和 构造函数继承 组合在一块
原型链实现对原型属性和方法的继承
借用构造函数实现对实例属性的继承

  • 拷贝继承

将父对象的 prototype 对象中的属性，拷贝给 Child 对象的 prototype 对象

定义函数的两种方式
1. 函数声明
function func() { // 函数体 }
2. 函数表达式 -> 把一个函数 赋值给一个变量，此时形成函数表达式
var 变量 = 匿名函数;

作用域
	局部变量 & 全局变量 ： 声明位置、作用域、生存期 三者不同
	隐式全局变量(可删除) & 全局变量(不可被删除)
块级作用域：JS没有块级作用域，只有函数除外

# 简单的闭包

闭包：可以简单理解为 定义在一个函数内部的函数
用途：
	可以读取函数内部的变量
	让这些变量的值始终保持在内存中
缺点：
	内存消耗
	会在父函数外部，改变父函数内部变量的值
